# vfmax

Benchmark comparing `vfloat4` element-wise max implementations on LoongArch with LSX. See [Godbolt](https://godbolt.org/z/qfEYc3j3x) for comparison of code generated by different compiler versions.

Built for discussion at:
<https://github.com/loongson-community/discussions/issues/120>

## Build & Run

```
make
./benchmark
```

## Result

Run result on 3C5000:

```
Variant                ns/op  M vfmax/s
---------------------------------------
gcc16 (vfmax_gcc16)     20.68        48    100000 pairs x 100 iters
gcc15 (vfmax_gcc15)     26.73        37    100000 pairs x 100 iters
clang (vfmax_clang)     14.72        68    100000 pairs x 100 iters
opt (vfmax_opt)          5.04       198    100000 pairs x 100 iters
```
